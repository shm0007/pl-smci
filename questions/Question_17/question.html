Scenario: April is a college undergraduate who has created her own blogging system to allow users to log on via the Internet and write text posts. The site communicates using a standard encrypted web connection. A user's posts are stored as text files in a folder named after their username, with a filename that matches the title of the post. Within the users' folder is also a file called _password.txt containing their password, used by the blog software during authentication. They can make new posts once they are logged in. Anyone can view old posts by entering the name of the old post.The figure below the question shows the file layout.

One day, April noticed a new post in her directory (dnd_is_boring.txt). She is certain that she did not make that post.

Question: From the following, choose the answer that explains the most likely way that this could have happened.

 Passwords can be read by anyone. A man-in-the-middle attack intercepted April's password. The database backend had a remote vulnerability. A stranger physically compromised the server that runs April's blog.
