Scenario: Max has many online accounts. Max wants to be very secure, so they create a random 64-character password containing many upper- and lower-case letters, many numbers, and many special characters (e.g., !, @, #, ?, etc.). No characters repeat adjacent to each other (e.g., "aa"), and no special character is used more than once. Max then memorizes and uses this very strong password for all their online accounts.

Question: Choose the option that describes the most significant security problem with this approach.

 Passwords can be leaked when systems are compromised. The password would be hard to memorize, so Max would probably write it down. The password might not meet a particular website's password rules. A computer algorithm could guess the password with enough tries.
